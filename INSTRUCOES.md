# Agendador de WhatsApp para Grupos 🚀

Este é um projeto de código aberto para agendar mensagens (texto, fotos e vídeos) em grupos de WhatsApp de forma totalmente gratuita.

## Tecnologias
- **Backend:** Node.js + Baileys (WhatsApp Web API)
- **Frontend:** React + Vite + Lucide Icons
- **Banco de Dados:** SQLite
- **Estilização:** CSS Vanilla (Glassmorphism Dark Mode)

## Como Usar

1. **Configuração Inicial:**
   Certifique-se de que o Node.js está instalado.

2. **Rodar o projeto:**
   No diretório raiz, execute:
   ```bash
   npm run dev
   ```

3. **Conectar o WhatsApp:**
   - Abra o navegador em `http://localhost:5173`.
   - Escaneie o QR Code que aparecerá na tela com o seu WhatsApp (Aparelhos Conectados).

4. **Agendar:**
   - Selecione o grupo desejado.
   - Escreva a mensagem ou selecione um arquivo (imagem/vídeo).
   - Escolha a data e hora.
   - Clique em "Agendar Mensagem".

## Observações
- O computador onde o backend está rodando precisa estar ligado na hora do agendamento.
- O uso excessivo para SPAM pode levar ao banimento do número. Use com responsabilidade.
